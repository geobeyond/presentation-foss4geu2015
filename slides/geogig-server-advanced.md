# Advanced commands

```bash
~/your-folder $ geogig ls HEAD:grids
# List the tree of a vector grids at HEAD commits

~/your-folder $ geogig show HEAD:grids
# Describe a given types of objects in a tree

~/your-folder $ geogig show HEAD:grids/1
# Print out the values of all attributes and their corresponding names from the feature type

~/your-folder $ geogig branch mybranch
# Create a new branch

~/your-folder $ geogig checkout mybranch
# Move to a new branch
```
