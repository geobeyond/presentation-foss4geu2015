# Quickstart commands


```bash
~$ geogig help
# List geogig commands

~/your-folder $ geogig init
# Initialise a geogig repository in your-folder

~/your-folder $ geogig shp import your_shapefile
# Import a .shp file (or try OSM, Spatialite or PostGIS)

~/your-folder $ geogig ls
# What's in your geogig repository

~/your-folder $ geogig status
# What you can commit to a snapshot
```
